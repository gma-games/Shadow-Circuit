## Tesztelés, fordítás és futtatás

### Tesztelés

- Új verzió publikálásához létre kell hozni egy új tag-et az alábbi formátumban: ``vx.x.x`` pl.: ``v0.0.1``
- Ez a művelet elindítja az automatikus EditMode és PlayMode teszteket, melyek eredményét feltölti az Action-s oldalra.

### Fordítás

- Amennyiben sikeresek a teszek, új elkezdi fordítani a három célplatformra: Windows, Linux MacOS
- A fordított állományt a GitHub [Releases](https://github.com/gma-games/Shadow-Circuit/releases)-re tölti fel

### Futtatás

- [Releases](https://github.com/gma-games/Shadow-Circuit/releases) oldalon le kell tölteni az operációs rendszernek megfelelő állományt, ki kell csomagolni és el is lehet indítani.
